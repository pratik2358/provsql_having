We start about preliminaries about (m-)semirings, annotated relations,
and extended relational algebra queries; we refer to
\cite{sen2026provsql} for a full description of the semantics of
extended relational algebra queries with terminal aggregation (i.e.,
without \sql{HAVING}).

\paragraph*{M-Semirings}

\begin{definition} \label{def:semiring}
  A \emph{semiring} is an algebraic structure $(\mathbb K,\oplus,\otimes,\mathbb 0,\mathbb 1)$ where $\mathbb K$ is a set, $\oplus$ and $\otimes$ are binary operations on $\mathbb K$, and $\mathbb 0,\mathbb 1\in \mathbb K$, such that:
  \begin{itemize}
    \item $(\mathbb K,\oplus,\mathbb 0)$ is a commutative monoid and $(\mathbb K,\otimes,\mathbb 1)$ is a (not necessarily commutative) monoid;
    \item $\otimes$ distributes over $\oplus$: $\forall~a,b,c\in K$, $a\otimes (b\oplus c) = (a\otimes b)\oplus (a\otimes c)$ and $(a\oplus b)\otimes c = (a\otimes c)\oplus (b\otimes c)$;
    \item $\mathbb 0$ is an annihilator for $\otimes$: $\forall a\in K$, $a\otimes \mathbb 0 = \mathbb 0 \otimes a = \mathbb 0$.
  \end{itemize}
\end{definition}

\begin{example}\label{ex:semirings}
  Some example semirings are:
  \begin{itemize}
    \item The counting semiring $\mathbb N$: $(\mathbb N,+,\times,0,1)$.
    \item The Boolean semiring $\mathbb B$: $(\{\top,\bot\},\lor,\land,\bot,\top)$.
    \item The Why-provenance semiring over a set of variables $X$: $({2^2}^X,\cup,\Cup,\emptyset,\{\emptyset\})$ where $\Cup$ is defined as follows: $A\Cup B := \{a\cup b \mid a\in A, b\in B\}$.
  \end{itemize}
\end{example}

We extend the definition of semiring with a monus operation $\ominus$;
note that this definition, when possible, uniquely defines such an
operation:
\begin{definition} \label{def:m-semiring}
  Let $(\mathbb K, \oplus, \otimes, \mathbb 0, \mathbb 1)$ be a semiring.
  For $a, b\in\mathbb K$, define
$a\le b$ as $\exists c\in\mathbb K, b = a\oplus c$; if $\leq$ is an order relation, it is called
the \emph{natural order} of the semiring, and the semiring is said to be
\emph{naturally ordered}. Note that $\leq$ is always a
pre-order, but not always antisymmetric. An m-semiring
is defined as a naturally ordered semiring in which we can introduce an
operation $\ominus$ satisfying the \emph{residuation law}:
\(
  x\ominus y \le z \iff x \le y\oplus z
  \).
\end{definition}

Not all semirings are naturally ordered, and it is possible to construct
a naturally ordered semiring where a $\ominus$ cannot be introduced
\cite{amarilli2016example}.
An m-semiring $(\mathbb K, \oplus, \otimes, \ominus, \mathbb 0, \mathbb
1)$ satisfies some basic properties, for $a, b, c\in\mathbb K$:
  \begin{itemize}
    \item \lean{SemiringWithMonus}{add_monus}
      $a\oplus(b\ominus a) = b\oplus(a\ominus b)$;
    \item \lean{SemiringWithMonus}{monus_add}
      $(a\ominus b)\ominus c = a\ominus(b\oplus c)$;
    \item \lean{SemiringWithMonus}{monus_self}
      \lean{SemiringWithMonus}{zero_monus} $a\ominus a = \mathbb 0\ominus a=\mathbb 0$.
  \end{itemize}

Additionally to the their definition, m-semirings may have other properties. Useful ones are:
\begin{itemize}
  \item \textbf{Commutativity of $\otimes$}: $\forall a,b\in \mathbb K$, $a\otimes b = b\otimes a$.
  \item \textbf{Idempotence}: \lean{SemiringWithMonus}{idempotent} $\forall a\in \mathbb K$, $a\oplus a = a$.
  \item \textbf{Absorptivity} \lean{SemiringWithMonus}{absorptive} $\forall a\in \mathbb K$, $\mathbb 1 \oplus a = \mathbb 1$.
  \item \textbf{Distributivity of $\otimes$ over $\ominus$}:
    \lean{SemiringWithMonus}{mul_sub_left_distributive} $\forall a,b,c\in
    \mathbb K$, $a\otimes (b\ominus c) = (a\otimes b) \ominus (a\otimes
    c)$.
\end{itemize}

In this work, we restrict to \emph{commutative} semirings, which is
actually the setting of most works on data provenance
\cite{green2007provenance,geerts2010database}, though not of
\cite{sen2026provsql}.
All semirings from Example~\ref{ex:semirings} are commutative
and can be extended to m-semirings; this is also the case for almost all
semirings that have been used in works about data provenance.
Table~\ref{tab:semirings} summarizes the definitions of the $\ominus$
operator and whether they satisfy the three other properties above for
the semirings in Example~\ref{ex:semirings}, along with the Boolean
function semiring \cite{green2007provenance,senellart2017provenance}, the
Which[$X$] or \emph{lineage} semiring \cite{green2017provenance}, the tropical semiring \cite{green2007provenance}, the Viterbi semiring
\cite{green2017provenance}, the security or access control semiring
\cite{green2017provenance}, the $\mathbb{N}[X]$ or How[$X$] semiring
\cite{green2007provenance} and the \L ukasiewicz semiring
\cite{gradel2005provenance}.
We note that only the Boolean, Boolean function, Tropical, Viterbi and \L
ukasiewicz semirings are idempotent, absorptive and have $\otimes$
distributive over~$\ominus$.
These properties will be important for the semantics of \sql{HAVING}
queries. Note that this table, along with the formal Lean proofs that
ensure its validity, allows
to fix in a definitive way a
few erroneous statements in the literature: \cite{geerts2010database} had
wrongly claimed that the tropical semiring over reals could not be equipped with a
$\ominus$; \cite{amsterdamer2011provenance} wrongly claimed that
both $\mathbb{N}[X]$ and Why[$X$] had a $\otimes$ distributive over $\ominus$
(called Axiom A13 in \cite{amsterdamer2011provenance}). We stress that
errors are bound to happen given some counter-intuitive properties of
these elementary building blocks of data provenance, and that it
is important to have clean, formally proven definitions and properties,
which is one of the contributions of this work.

For queries with grouping and aggregation, an additional useful notion
\cite{amsterdamer2011provenance} is that of \emph{$\delta$-semirings}, which is a semiring with an additional unary operator $\delta:\mathbb K\to \mathbb K$ having the following two properties: (i) $\delta(\mathbb 0)=\mathbb 0$ and (ii) $\delta(\mathbb 1 \oplus \dots \oplus \mathbb 1)=\mathbb 1$.
Intuitively, $\delta$ is used to capture the semantics of the
existence of a group. For reasons we will explain later on, we fix the
definition of $\delta$ to be $\delta(x)=x$ in
idempotent semirings (in which $\mathbb 1\oplus\dots\oplus\mathbb
1=\mathbb 1$); and $\delta(x)=\mathbb 1$ for all $x\neq\mathbb 0$ in
non-idempotent semirings.

\paragraph*{Useful results on m-semirings.} We discuss now some
additional results on m-semirings that we use later. First we show that idempotence is equivalent to $\ominus$ being right-distributive over $\oplus$.
\begin{propositionrep}\label{prop:idempotent_iff_add_monus}
  \lean{SemiringWithMonus}{idempotent_iff_add_monus}
  An m-semiring $\mathbb K$ is idempotent if and only if $\ominus$
  is right-distributive over~$\oplus$ in $\mathbb K$, i.e., iff $\forall a, b,
c\in\mathbb K, (a\oplus b)\ominus c=(a\ominus c)\oplus (b\ominus c)$.
\end{propositionrep}
\begin{appendixproof}
  We show both implications.

  \textbf{($\Rightarrow$)}
  \lean{SemiringWithMonus}{add_monus_of_idempotent}
  Assume $\mathbb K$ to be idempotent.

  Consider $a,b \in\mathbb K$ such as $a\leq b$. Then $\exists c, a+c=b$.
  So $a \oplus b = a \oplus (a \oplus c)=(a\oplus a)\oplus c=a \oplus c = b$.
  Conversely, if $a\oplus b = b$, we trivially have $a\leq b$.

  Now consider an element $u\in\mathbb K$ such that $a \leq u$ and $b
  \leq u$, i.e.,
  $a \oplus u = u$ and $b\oplus u =u$. Observe that $a\oplus b\oplus
  u=a\oplus u=u$, so that $a\oplus b\leq u$.

  But since $a\leq a \oplus b$ and $b \leq a \oplus b$,
  $a \oplus b $ is the least such $u$ for which $a\leq u $ and $b \leq u$.
  In other words, $\oplus$ is the join of the semilattice $(\mathbb K,\leq)$.

  \medskip
  We show $(a\oplus b)\ominus c=(a\ominus c)\oplus (b\ominus c)$ by
  showing both inequalities.

  (i) By definition of monus : $x \ominus y \leq z \iff x \leq y \oplus
  z$. Note that $a\leq a\oplus b$ and $a\oplus b\leq c \oplus ((a\oplus
  b)\ominus c)$ by applying the residuation law to the inequality
  $(a\oplus b)\ominus c\leq (a\oplus b)\ominus c$. This means $a\leq
  c\oplus((a\oplus b)\ominus c)$ which implies $a\ominus c\leq (a\oplus b)\ominus c$.
  Similarly, $b\ominus c\leq (a\oplus b)\ominus c$. By using the fact
  that $\oplus$ is the join, we get
  $(a\ominus c)\oplus (b\ominus c) \leq (a\oplus b) \ominus c$.

  (ii) We have $a\leq c \oplus (a\ominus c)$ and $b\leq c\oplus (b\ominus c)$. So, $a\oplus b \leq c\oplus c \oplus (a\ominus c) \oplus (b\ominus c)$.
  By idempotence of $\oplus$,  $a\oplus b \leq c \oplus (a\ominus c)
  \oplus (b\ominus c)$, or by the residuation law
$(a\oplus b) \ominus c \leq (a\ominus c)\oplus (b\ominus c) $.

\bigskip
\textbf{($\Leftarrow$)}
  \lean{SemiringWithMonus}{idempotent_of_add_monus}
  Assume that $\ominus$ is right-distributive over $\oplus$.
For any arbitrary $a\in \mathbb K$, we thus have:
\[
  (a\oplus a)\ominus a
  = (a\ominus a)\oplus(a\ominus a).
\]
Since $a\ominus a=\mathbb 0$, the right-hand side is $\mathbb 0$. And
$(a\oplus a)\ominus a = \mathbb 0\implies a\oplus a\leq a$ by the
residuation law.
  But, trivially, $a\le a\oplus a$ so the antisymmetry of $(\mathbb K,\leq)$ gives $a\oplus a=a$.
  Thus, \(\forall a\in\mathbb K, a \oplus a=a.\)
\end{appendixproof}
\begin{corollaryrep}\label{lem:finite-add-monus}
Let \(\mathbb K\) be an
idempotent
m-semiring.
Then \(\forall n \ge 1\) and \(\forall a_1,\dots,a_n,c \in K\),
\(
\left(\bigoplus_{i=1}^n a_i\right)\ominus c
=
\bigoplus_{i=1}^n (a_i \ominus c).
\)
\end{corollaryrep}

\begin{appendixproof}
We proceed by induction on \(n \ge 1\).

For \(n=1\), the identity is trivial.

Assume the identity holds for \(n-1\).
Then
\begin{align*}
\Bigl(\bigoplus_{i=1}^n a_i\Bigr)\ominus c
&= \Bigl(\bigl(\bigoplus_{i=1}^{n-1} a_i\bigr)\oplus a_n\Bigr)\ominus c \\
&= \Bigl(\bigoplus_{i=1}^{n-1} a_i\Bigr)\ominus c \;\oplus\; (a_n\ominus c) \\
&= \bigoplus_{i=1}^{n-1} (a_i\ominus c) \;\oplus\; (a_n\ominus c)
= \bigoplus_{i=1}^n (a_i\ominus c),
\end{align*}
where the second equality uses
Proposition~\ref{prop:idempotent_iff_add_monus} and the third uses the
induction hypothesis.
\end{appendixproof}

Finally, we isolate two elementary inequalities on the residual
characterization of $\ominus$ in m-semirings.
\begin{lemmarep}\label{lem:x-y<=x}
  \lean{SemiringWithMonus}{monus_le}
  In any m-semiring $\mathbb K$,
$\forall x,y\in \mathbb K$, $x\ominus y \le x$.
\end{lemmarep}

\begin{appendixproof}
  By the residuation law (see Definition~\ref{def:m-semiring}) with $z=x$, we have
\[
x\ominus y \le x \quad\Longleftrightarrow\quad x \le y\oplus x.
\]
But $x\le y\oplus x$ holds by definition of $\leq$ (and commutativity of
$\oplus$).
\end{appendixproof}

\begin{lemmarep}\label{lem:x<=y+(x-y)}
  \lean{SemiringWithMonus}{le_plus_monus}
  In any m-semiring $\mathbb K$,
  $\forall x,y\in \mathbb K$, $x \le y\oplus (x\ominus y)$.
\end{lemmarep}

\begin{appendixproof}
  Apply the residuation law (see Definition \ref{def:m-semiring}) with $z=x\ominus y$:
since $x\ominus y\le x\ominus y$ holds trivially, we obtain
\[
x \le y\oplus (x\ominus y),
\]
as required.
\end{appendixproof}

\paragraph*{Annotated relations and semantics}

In this paper, we use the notion of annotated relations detailed in \cite{sen2026provsql}(Sec.\ IV), defined as follows.
Given an algebraic structure $\mathbb K$ and a set of $\mathcal V_{\mathbb K}\supseteq \mathcal V$ of values (which can be either $\mathcal V$ or an extension of it) a $\mathbb K$-relation of arity $k\in\mathbb N$ over $\mathcal V_{\mathbb K}$ is a multiset of tuples over $\mathcal V_{\mathbb K}^k\times\mathbb K$ of the form $(u,\alpha)$ with $u\in \mathcal V_{\mathbb K}^k$ and $\alpha\in \mathbb K$.
We call $\alpha$ the annotation of the tuple $u$.

A $\mathbb K$-instance, denoted as $\hat I$, over a database schema $\mathcal D$ is a function mapping each relation symbol $R$ of $\mathcal D$ to a $\mathbb K$-relation over $\mathcal V_{\mathbb K}$ of the same arity as $\mathcal D(R)$.
We consider, instead of attribute name, their \emph{positional index}, denoted by $\#i$ for $i\in \mathbb N^*$.
A \emph{term} is then an expression involving values from $\mathcal V$ and positional indices, and the \emph{max-index} of a term is the largest positional index appearing in it ($0$ if none appear).

The extended relational algebra semantics for a query $q$ over annotated relations, denoted as $\predsem{\hat I}{}{q}$, is defined for each operator of the relational algebra over $\mathbb K$-relations \cite{sen2026provsql}.
There, semantics of aggregation queries were defined only when aggregation was the last operator of the query.
In this paper, we extend the semantics of the selection operator to the case where selection is applied to aggregate expressions (\sql{HAVING} conditions), either by comparing an aggregate with a constant or an aggregate with an aggregate.
